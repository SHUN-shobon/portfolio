# [s2n.tech](https://s2n.tech)

My personal portfolio website.

## Development

### Prerequisites

- Node.js (The version is specified in `.tool-versions`)
- pnpm (The version is specified in `package.json#packageManager`)

### Setup

```sh
pnpm install
```

### Development server

```sh
pnpm dev
```

### Build

```sh
pnpm build
```
